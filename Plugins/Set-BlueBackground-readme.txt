
Set-BlueBackground.ps1
Simple example of updating a page with a plugin by demonstrating how to set
the page background.

--------------------------------------------------------------------------------

0. Save Set-BlueBackground.ps1 to your local drive in some <path>
1. From the OneMore More menu, choose Run Plugin
2. Enter a name if you want to save this as a favorite such as "Set-BlueBackground"
3. Enter the command: powershell
4. Enter arguments: -file <path>\Set-BlueBackground.ps1
5. Set the Target field to 'Current page'
6. Click Save to save on the File/Plugins menu
7. Click Run to run now or activate from the File/Plugins menu
